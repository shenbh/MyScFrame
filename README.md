### 10.全面屏/刘海屏适配
```
        <!--适配华为刘海屏-->
        <meta-data
            android:name="android.notch_support"
            android:value="true"/>
        <!--适配小米刘海屏-->
        <meta-data
            android:name="notch.config"
            android:value="portrait|landscape"/>

        <!--适配全面屏-->
        <meta-data
            android:name="android.max_aspect"
            android:value="2.4"/>
```
